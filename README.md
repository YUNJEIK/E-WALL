# E-WALL

Proyecto web de un muro social interactivo para eventos.

## Descripción

E-WALL es una plataforma pensada para eventos donde los invitados pueden compartir fotos, mensajes o contenido que se proyecta en tiempo real en una pantalla durante el evento.

El objetivo es generar una experiencia interactiva entre los asistentes y el evento.

## Tecnologías utilizadas

- HTML5
- CSS3
- SASS
- Bootstrap
- Git
- GitHub
